# Aprendizagem — Desenvolvimento Web

Repositório de estudo pessoal, fase inicial de
aprendizagem de programação web. Sem projeto associado por agora —
o objetivo é praticar HTML, CSS, JavaScript, SQL e, mais tarde, Node.js.

## Organização
- `html-css/` — exercícios de HTML e CSS
- `javascript/` — exercícios de JS puro
- `sql/` — exercícios de SQL/MySQL
- `notas/diario.md` — registo de progresso e dúvidas

## Como usar
Cada pasta de exercício tem ficheiros independentes.
Para HTML/CSS, basta abrir o `.html` no browser.
